В этой папке будет организовано взаимодействие с mobx и хранение данных
MobX - это библиотека для управления состоянием приложения в JavaScript, которая часто используется в связке с React. 
Она предоставляет механизмы для облегчения отслеживания изменений состояния и автоматического обновления компонентов при изменении этого состояния.
